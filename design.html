🔍 The New Report

What Looks Different in UAD 3.6

Five key changes agents should understand to navigate the redesigned appraisal report.

1

Dynamic URAR Replaces Legacy Form Layouts

The redesigned URAR presents sections based on what is relevant to the assignment and property. Less time locating information across different legacy form formats.

2

More Standardized Data Fields = Cleaner Corrections

Increased structured reporting and defined inputs mean underwriters and reviewers can validate key details faster and point corrections to specific sections.

3

Submission Summary Report Becomes a Sharper Fix List

Feedback aligns with the redesigned URAR structure. You will see more precise resolve-this-item requests tied to exact report sections.

4

Additional Properties Analyzed Not Used Table

A dedicated table where appraisers log transactions they considered and screened out — with fields for address, sale date/status, reason not used, and comments. This makes ROVs dramatically more transparent.

5

Two New Sections for Upgraded Homes

Section 5: Disaster Mitigation and Section 6: Energy Efficient and Green Features use pick-list structures. If no relevant features apply, the section stays minimal.

🎯

Agent Takeaway

Suggested comps in an ROV can now be evaluated and documented transparently in the report, so the lender/underwriter sees the appraiser logic without a long back-and-forth.

ROV Workflow — Easier Under UAD 3.6

ROVs work best when they focus on factual corrections and market-supported transactions. UAD 3.6 helps because the Additional Properties Analyzed Not Used table lets you see what the appraiser already considered.

🗺

Your ROV Roadmap

Step 1: Check if your suggested comp is already listed as analyzed-not-used. If it is, read the stated reason and decide if you have a fact correction or need a different comp.
Step 2: If it is not listed, submit it as additional market support and request it be evaluated.
Step 3: Keep it tight — 2–5 comps max, plus documentation for the exact facts you want reviewed.

✉️ ROV Request Template (UAD 3.6-Aware)
Subject: Reconsideration of Value Request — Additional Property Details / Market Data (UAD 3.6)

Hello,

Please review the attached documentation for [Property Address]. We are requesting consideration of:

Property detail clarification: [specific detail] supported by [permit/receipt/photo/solar agreement].

Additional market support: The attached transactions at [addresses] reflect [same neighborhood tier / similar GLA / similar quality / similar energy features].

Key similarities: [detail]  |  Key differences: [detail]

If any suggested transactions are determined to be less comparable, please consider documenting them in the Additional Properties Analyzed Not Used section with the reason and brief comment so the analysis is clear in the report structure.

🌿 Green Energy + Resiliency

How to Get Full Credit

UAD 3.6 adds Section 6 (Energy Efficient and Green Features) and Section 5 (Disaster Mitigation) with structured pick lists. When features are present and documented, they are easier to capture consistently.

The Green Documentation Bundle

1 — Solar / Renewable

Solar and Renewable Energy

  • Ownership type: Owned / Leased / PPA
  • System size (kW), install date, warranties
  • Inverter/battery details (if applicable)
  • Contract terms for leased/PPA systems
2 — Certifications

Certifications and Ratings

  • HERS report or ENERGY STAR documentation
  • Other recognized program certificates
  • Issuer + date + homeowner docs bundle
3 — Efficiency

Efficiency Upgrades

  • HVAC efficiency specs + install date
  • Insulation/windows — type + year
  • Water heater type
  • Air sealing/envelope improvements
  • Optional: utility summaries

Resilience Upgrades (Section 5: Disaster Mitigation)

Document items that reduce risk and impact. What is relevant varies by market:

🏠 Roof Rating 🪟 Impact-Resistant Openings 🌲 Defensible Space ⛈️ Storm Features 🌊 Flood Mitigation 📄 Certification Paperwork
⚠️ Watch-Out

Visibility Does Not Equal Automatic Value

Green features gain visibility through structured capture, but market impact still depends on credible market support and analysis. Documentation and comp selection strategy matter.

⚠️ Watch-Out

Solar Ownership Changes Buyer Reaction

Owned vs. leased vs. PPA systems affect how buyers respond in many markets. Provide solar agreements early so the analysis aligns with actual buyer behavior.
