# Rutas — Año Sabático África 2027

## Ruta A — Corredor Noroccidental (mejor relación costo/francés)

```
Marruecos → Mauritania → Senegal → Túnez
```

Países de lengua francesa, costos moderados, sin visa o visa fácil para pasaporte colombiano. Cubre el Magreb y África Occidental francófona.

---

## Ruta A Extendida — RECOMENDADA (objetivo UFBA)

```
Marruecos → Mauritania → Senegal → Cabo Verde → Benin → Ghana → (vuelta)
```

**Por qué esta ruta:**
- Cubre el triángulo completo de la diáspora afrobrasileña
- Benin (Ouidah) es la fuente directa del Candomblé Jeje — no tiene reemplazo
- Ghana no requiere visa para colombianos
- Cabo Verde sin visa y estratégicamente ubicado entre Senegal y el Golfo de Benín
- Maximiza el valor académico para UFBA sin aumentar drásticamente los costos

**El triángulo del Candomblé:**
| Región | País | Grupo étnico | Tradición en Brasil |
|--------|------|-------------|---------------------|
| África Occidental (Sahel/Costa) | Senegal | Wolof, Mandinka | Malê / Islam en Bahía |
| Golfo de Benín | Benin/Togo | Fon, Ewe | Candomblé Jeje |
| Costa de Oro | Ghana/Nigeria | Ashanti, Fante | Mina / Candomblé Keto |
| Yoruba | Nigeria/Benin | Yoruba (Nagô) | Candomblé Keto (orixás) |

---

## Ruta B — Circuito África Oriental (más fácil logísticamente)

```
Rwanda → Uganda → Kenya
```

Un solo visado (East African Tourist Visa, $100, 90 días). Más turístico, menos relevante para el eje académico UFBA. Reservar para un viaje posterior o como extensión si queda tiempo.

## Correcciones vs. plan anterior

- Túnez **NO es VOA** para colombianas independientes (visa consular ~$95, tramitar desde Brasil)
- Senegal es más caro de lo estimado (housing 2x Marruecos, +60% costo de vida)
- Mauritania **sí es VOA**: €55/$60 al llegar — puente perfecto entre Marruecos y Senegal
- Malí y Níger: **EXCLUIR** por seguridad (fronteras Argelia cerradas, riesgo terrorismo)
- Túnez puede reintegrarse si se tramita la visa desde São Paulo antes de salir
