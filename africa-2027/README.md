# Año Sabático África 2027

**Ventana:** Enero – Septiembre 2027  
**Duración:** 8–9 meses  
**Budget target:** ~$900–1,200 USD/mes  
**Pasaporte:** Colombiano | **Perfil:** Nómada digital, mujer sola, francés

## Objetivo

Llegar a UFBA 2028 con base sólida en historia africana y conexiones con la diáspora afrobrasileña. El viaje es tanto personal como académico: rastrear la ruta del Atlántico Negro — de los imperios africanos a los terreiros de Bahía.

## Ruta recomendada

**Ruta A Extendida — Corredor Noroccidental + Golfo de Benín**

```
Marruecos → Mauritania → Senegal → Cabo Verde → Benin → Ghana → (vuelta)
```

Esta ruta cubre el triángulo completo del Candomblé:
- **Senegal** → Malê/Islam (wolof, mandinga)
- **Benin** → Jeje/Vodún (fon, ewe) — fuente directa de Ouidah
- **Ghana** → Mina/Ashanti (fante, ashanti)

## Contexto: Hipótesis B

Si el ingreso a UFBA (PEC-G) es enero 2028, el año 2027 queda libre después de Centroamérica (que termina dic 16 2026). Verificar resultado Celpe-Bras 18-dic-2026 para activar este plan.
