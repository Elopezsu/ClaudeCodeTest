## Reglas de búsqueda

### Prioridad de fuentes (de mayor a menor confianza)
1. **FamilySearch registros históricos** — bautismos, matrimonios, defunciones colombianos
2. **FamilySearch árbol** — perfiles públicos con fuentes citadas
3. **Archivos parroquiales digitalizados** — especialmente Santa Rosa de Osos (registros desde 1718)
4. **Google + genealogía Colombia** — para contexto y pistas
5. **Wikipedia / historia regional** — para contexto de migración y La Violencia

### Reglas de confianza
- **Alta:** coincide nombre + apellido + lugar + año aproximado + tiene fuente documental
- **Media:** coincide nombre + apellido + lugar, sin fuente documental directa
- **Baja:** solo coincide nombre parcial o lugar
- Solo actualizar el árbol con confianza **alta o media**

### Contexto histórico clave
- **La Violencia (1948-58):** afectó fuertemente el Bajo Cauca antioqueño. Causa de migración y silencio transgeneracional. Los "Pájaros" (sicarios conservadores) actuaron en zonas liberales.
- **Margento** = corregimiento de Caucasia, Bajo Cauca. Diócesis Santa Rosa de Osos.
- **Prasca** = apellido italiano, inmigrante llegó por Barranquilla s.XIX. Buscar en Archivo Histórico del Atlántico.
- **Zabaleta** = apellido más fuerte en Cesar y costa Caribe.
- **Bautismo retrasado 22 años** del padre (nació 1954, bautizado 1976) = posible desplazamiento.

### Qué buscar en cada persona
Para cada gap, buscar:
1. Nombre de padres (para extender el árbol)
2. Fecha y lugar de nacimiento exactos
3. Fecha y lugar de matrimonio
4. Hermanos (para confirmar identidad)
5. Nombre de padrinos de bautismo (fuente adicional)

## Formato de hallazgo
Siempre documentar:
- Nombre completo encontrado
- Fecha/lugar
- Nombre de la fuente (colección FamilySearch, URL, libro/folio si aplica)
- Nivel de confianza
